# python-svg-generater

pythonを使って、cui操作でsvg画像を生成します


## ファイル構成

    .
    ├── README.md
    ├── __pycache__
    │   ├── funcs.cpython-36.pyc
    │   ├── main.cpython-36.pyc
    │   └── write.cpython-36.pyc
    ├── main.py
    ├── main2.py
    ├── funcs.py
    └── write.py

## 環境構築

python3をインストール


## 機能

### ターミナルでの対話入力で生成
```
$ cd python-svg-generater
$ python main.py
```


### エディタ操作で生成
```
$ cd python-svg-generater
$ python main2.py
```
